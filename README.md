# Keyboard CSS

## Getting Started

Simplest way to use this is through link tag (thanks to <https://unpkg.com/>):

```html
<link rel="stylesheet" href="https://unpkg.com/keyboard-css/css/main.min.css" />
```

You can also install it from npm:

```bash
npm i keyboard-css
```
